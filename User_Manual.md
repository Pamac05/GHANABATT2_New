# User Manual - GHANBATT2 Website

## Page 1: Getting Started

### Welcome to GHANBATT2 Website
This user manual provides comprehensive guidance for navigating and utilizing the GHANBATT2 website, the official digital platform for the Ghanaian Battalion's peacekeeping mission in Abyei.

### System Requirements
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection (minimum 3G speed)
- Screen resolution: 1024x768 or higher
- JavaScript enabled

### Accessing the Website
1. Open your web browser
2. Navigate to: https://pamac05.github.io/GHANABATT2_New/
3. The homepage will load automatically

## Page 9: Troubleshooting

### Common Issues

#### Images Not Loading
**Problem**: Images appear as broken links
**Solutions**:
1. Check internet connection
2. Clear browser cache
3. Disable ad blockers temporarily
4. Try a different browser

#### Navigation Not Working
**Problem**: Menu items not responding
**Solutions**:
1. Enable JavaScript in browser settings
2. Update browser to latest version
3. Check browser console for errors
4. Try refreshing the page

#### Mobile Display Issues
**Problem**: Site not displaying correctly on mobile
**Solutions**:
1. Rotate device to landscape mode
2. Update mobile browser
3. Clear mobile browser cache
4. Check mobile data connection

### Performance Issues

#### Slow Loading Times
**Causes and Solutions**:
- **Slow Internet**: Check connection speed
- **Browser Issues**: Clear cache and cookies
- **High Traffic**: Try accessing during off-peak hours
- **Device Performance**: Close other applications
